## 概述

本报告基于蒙特卡洛模拟方法，对"贪骰无厌"游戏进行了全面的数据分析。模拟考虑了：
- 6个骰子分两组的投掷机制（每组3个骰子求和）
- 16个列（3-18）的不同高度和概率
- 遭遇、陷阱、道具的效果
- 三种运气情况（最佳/普通/最差）

**模拟规模**：每种情况 3000-10000 局游戏

**修正说明**：本报告基于修正后的模拟逻辑，正确反映游戏规则：每次投骰选择一种分组方式，每个数值对应的列前进1格。

## 骰子概率分析

6个骰子分成两组（各3个），每组求和得到列号。各列被投中的概率如下：

| 列号 | 概率 | 说明 |
|:---:|:---:|:---|
| 10, 11 | **80.2%** | 最容易投中 |
| 9, 12 | 76.3% | 非常容易 |
| 8, 13 | 69.4% | 较容易 |
| 7, 14 | 56.6% | 中等 |
| 6, 15 | 41.1% | 较难 |
| 5, 16 | 28.0% | 困难 |
| 4, 17 | 14.8% | 很困难 |
| 3, 18 | **6.2%** | 极难投中 |

**关键发现**：列10/11的投中概率是列3/18的 **13倍**

### 为什么列3/18会出现超过1000积分的极端消耗？

从累积分布图可以看出：
1. 列3/18的95%分位数达到970-1030积分
2. 极端情况下（最倒霉的1-2%玩家）可能消耗超过1500积分
3. 原因：6.2%的概率意味着平均每16次投骰才命中1次
4. 连续50+次投不中的情况虽然罕见，但确实会发生

## 最佳策略建议

### 推荐登顶组合

**最佳3列组合**：列11 + 列10 + 列12
- 预计总净消耗：约 **101积分**（普通运气）
- 这三列概率高、消耗低

**最差3列组合**：列3 + 列4 + 列18
- 预计总净消耗：约 **1194积分**（普通运气）
- 差距：**约12倍**

### 策略总结

1. **优先攻略中间列**（9-13）
   - 概率高（69-80%）
   - 单列消耗低（6-100积分）

2. **避开边缘列**（3-5, 16-18）
   - 概率低（6-28%）
   - 消耗极高（106-485积分）

3. **利用事件收益**
   - 好运气时事件可以提供400+积分收益
   - 差运气时事件会额外扣除120+积分

1. **列选择至关重要**：最佳组合与最差组合相差约12倍消耗

3. **运气影响显著**：好运与坏运相差579积分
4. **推荐策略**：优先攻略列11、10、12，避开列3、4、18
