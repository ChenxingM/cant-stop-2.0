## 游戏进行

| 指令 | 说明 | 示例 |

## 奖励领取

| 指令格式 | 说明 | 示例 |
|----------|------|------|
| `领取{类型}奖励{数量}` | 领取打卡奖励 | `领取草图奖励1` |
| `领取{类型}奖励{数量}*{倍数}` | 领取双倍奖励 | `领取草图奖励1*2` |
| `我超级满意这张图{数量}` | 领取超常发挥奖励（+30分/张） | `我超级满意这张图1` |
| `数列{列号}登顶` | 领取列登顶奖励 | `数列7登顶` |
| `支线{编号}领取` | 领取支线任务奖励 | `支线1领取` |
| `主线{编号}领取` | 领取主线任务奖励 | `主线1领取` |

**奖励类型**：
- `草图` - 20积分/张
- `精致小图` - 80积分/张
- `精草大图` - 100积分/张
- `精致大图` - 150积分/张
- `超常发挥` - 30积分/张

### 特殊效果指令

某些遭遇/陷阱会给予特殊能力，使用以下指令触发：

| 指令格式 | 说明 | 示例 |
|----------|------|------|
| `使用上轮骰子:{值1},{值2},{值3}` | 使用上轮骰子值替换本轮 | `使用上轮骰子:3,4,5` |

| `骰子加3:{位置}` | 指定位置骰子+3 | `骰子加3:2` |

## 自定义口令

自定义口令由 GM 在管理界面或配置文件中配置，使用**精确匹配**。

### 配置文件

位置：`data/custom_commands.json`

格式：
```json
{
  "commands": [
    {
      "keyword": "领取圣诞礼物",
      "response": "恭喜领取成功！",
      "score_reward": 100,
      "per_player_limit": 1,
      "enabled": true
    }
  ]
}
```

| 字段 | 说明 |
|------|------|
| `keyword` | 触发关键词（精确匹配） |
| `response` | 回复消息 |
| `score_reward` | 积分奖励 |
| `per_player_limit` | 每人限制次数（0=无限） |
| `enabled` | 是否启用 |

### GM 界面管理

在 GM 界面的"📣 口令管理" Tab 中可以：
- 添加/编辑/删除口令
- 启用/禁用口令
- 导入/导出配置文件

## 标点转换说明

以下全角标点会自动转换为半角：

| 全角 | 半角 |
|------|------|
| `，` | `,` |
| `：` | `:` |
| `；` | `;` |
| `！` | `!` |
| `？` | `?` |
| `（` | `(` |
| `）` | `)` |
| `【` | `[` |
| `】` | `]` |

因此 `选择阵营：收养人` 和 `选择阵营:收养人` 效果相同。

## 快速参考

### 完整游戏流程

```
1. 选择阵营:收养人     ← 选择阵营
2. 轮次开始            ← 开始轮次
3. .r6d6              ← 投掷骰子
4. 7,11               ← 记录数值（移动标记）
5. 重复 3-4 直到无法继续或满意
6. 替换永久棋子        ← 保存进度
```
