# 机器人指令手册

## 指令说明

- `【】`：用于标识指令，**玩家实际输入时不需要输入【】符号**
- `▸`：机器人测算/提醒
- `▹`：机器人测算/子选项
- `*`：指令功能说明

**示例：**
- 文档中写的：`【轮次开始】`
- 玩家实际输入：`轮次开始`

---

## 起始指令

### 阵营选择

**指令:** `【选择阵营：xxx】`

**参数:** 
- `xxx` = 收养人 或 Aeonreth

**输出:**
```
您已选择阵营：xxx，祝您玩得开心～
```

---

### 帮助

**指令:** `【help】`

**功能:** 查看帮助信息

---

## 游戏进行指令

### 1. 轮次开始

**指令:** `【轮次开始】`

**功能:** 开始新一轮次

**输出:**
```
新轮次已开启
```

---

### 2. 投掷骰子

**指令:** `【.r6d6】`

**功能:** 进行一回合投掷（投掷6个6面骰子）

**输出:** 显示骰子结果

---

### 3. 记录数值（双标记）

**指令:** `【a,b】`

**功能:** 记录得到的两个数值a和b（需要玩家自己选择并计算）

**输出:**
```
玩家选择记录数值：
当前位置：；剩余可放置标记：
```

---

### 4. 记录数值（单标记）

**指令:** `【a】`

**功能:** 当只有一个标记可放置/移动时，记录得到的数值a

**输出:**
```
玩家选择记录数值：
当前位置：；剩余可放置标记：
```

**▸报错提醒1:**
```
检测到您仍有可放置/移动的临时标记，请进行移动。
```

**▸报错提醒2:**
```
检测到您当前数值无法移动，请重新设置。
```

---

### 5. 主动结束轮次

**指令:** `【替换永久棋子】`

**功能:** 玩家主动结束当前轮次

**输出:**
```
本轮次结束。
当前永久棋子位置：
进度已锁定，请打卡后输入【打卡完毕】恢复开启新轮次功能
```

---

### 6. 被动结束轮次

**指令:** `【进度回退】`

**功能:** 玩家自行计算无可移动标记后失败被动停止当前轮次

**输出:**
```
本轮次结束。
当前永久棋子位置：
```

---

### 7. 查看当前进度

**指令:** `【查看当前进度】`

**功能:** 查看玩家当前在地图上的所有坐标

**输出:**
```
当前位置：；剩余可放置标记：
当前永久棋子位置：
已登顶棋子数：
```

---

### 8. 打卡完毕

**指令:** `【打卡完毕】`

**功能:** 恢复开启新轮次功能

**输出:**
```
您可以开始新的轮次了～
```

---

### 9. 查看背包

**指令:** `【查看背包】`

**功能:** 查看当前积分、道具和物品

**输出:**
```
当前积分：
历史获得积分：
当前道具：
当前隐藏道具/物品：
```

---

### 10. 成就一览

**指令:** `【成就一览】`

**功能:** 查看玩家所有获得过的成就

---

## 奖励领取指令

### 1. 普通打卡奖励

**指令:** `【领取（类型）奖励n】`

**功能:** 领取普通打卡奖励

**输出:**
```
您的积分+
```

---

### 2. 打卡附加奖励

**指令:** `【我超级满意这张图n】`

**功能:** 打卡附加奖励（n为图的数量）

**输出:**
```
您的积分+(n*x)
```

# 不同类型的作品奖励
```
add_score 草图           # +20积分
add_score 精致小图       # +80积分
add_score 精草大图       # +100积分
add_score 精致大图       # +150积分
add_score 超常发挥       # +30积分
```

---

### 3. 特殊遭遇额外奖励

**指令:** `【领取（类型）奖励n*2】`

**功能:** 特殊遭遇额外奖励（双倍）

**输出:**
```
奖励已翻倍，您的积分+
```

---

### 4. 登顶奖励

**指令:** `【数列x登顶】`

**说明:** 请在一列登顶后立即输入此指令

**输出:**
```
恭喜您在【x】列登顶～
已清空该列场上所有临时标记。

✦登顶奖励
恭喜您获得

✦首达奖励（首次登顶时）
恭喜您在该列首次登顶，获得
```

---

## 地图内容物使用指令

### 道具商店

**指令:** `【道具商店】`

**功能:** 可查看并购买当前全地图已解锁道具

---

### 丑喵玩偶相关

#### 购买丑喵玩偶

**指令:** `【购买丑喵玩偶】`

**价格:** 150积分（常驻商品）

**说明:** 购买后可使用指令【捏捏丑喵玩偶】随机获得一定奖励

---

#### 捏捏丑喵玩偶

**指令:** `【捏捏丑喵玩偶】`

**限制:** 每天限3次

**随机结果:**

| 概率 | 结果 |
|------|------|
| 大概率 | 玩偶发出了吱吱的响声，并从你手中滑了出去 |
| 小概率 | 玩偶发出了呼噜呼噜的响声，似乎很高兴，你获得r3d6的积分 |

---

### 道具解锁

**指令:** `【添加xxx到道具商店】`

**功能:** 如果您解锁地图上的xxx道具，请立即使用此指令

## 指令速查表

### 基础操作

| 指令 | 功能 |
|------|------|
| `【选择阵营：xxx】` | 选择阵营（收养人/Aeonreth） |
| `【help】` | 查看帮助 |
| `【轮次开始】` | 开始新轮次 |
| `【.r6d6】` | 投掷6个骰子 |
| `【a，b】` | 记录两个数值 |
| `【a】` | 记录单个数值 |
| `【替换永久棋子】` | 主动结束轮次 |
| `【进度回退】` | 被动结束轮次 |
| `【打卡完毕】` | 恢复新轮次功能 |

### 查询操作

| 指令 | 功能 |
|------|------|
| `【查看当前进度】` | 查看地图坐标 |
| `【查看背包】` | 查看积分和道具 |
| `【成就一览】` | 查看所有成就 |
| `【道具商店】` | 查看道具商店 |

### 奖励操作

| 指令 | 功能 |
|------|------|
| `【领取（类型）奖励n】` | 普通打卡奖励 |
| `【我超级满意这张图n】` | 附加奖励 |
| `【领取（类型）奖励n*2】` | 双倍奖励 |
| `【数列x登顶】` | 登顶奖励 |

### 特殊操作

| 指令 | 功能 |
|------|------|
